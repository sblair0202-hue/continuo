# Sprint 1: Core Backend and Intelligence Loop

## Goal

Build the minimum backend needed to accept a transcript, extract structured field intelligence, and save reviewed records.

## Deliverables

- FastAPI app
- SQLAlchemy models
- Pydantic schemas
- SQLite local database
- Transcript ingestion endpoint
- AI extraction stub
- Approval endpoint
- Account creation
- Contact creation
- Activity creation
- Task creation

## Do Not Build Yet

- Real mobile app
- Real Salesforce integration
- Real Gmail/calendar integration
- Real audio transcription
- Manager dashboard

## Acceptance Criteria

- Can paste a transcript and get structured output.
- Can approve output and save records.
- Can retrieve accounts, activities, contacts, and tasks.
